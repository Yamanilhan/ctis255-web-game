# ctis255-web-game
CTIS 255 - Interactive Web Game
Bu proje Bilkent Üniversitesi Bilişim Sistemleri ve Teknolojileri bölümü bünyesindeki CTIS 255 dersi için geliştirilmiştir.

Geliştiriciler: Yaman İlhan, Demir Dereli, Berke Mehmet Aksoy.

Özellikler:

JavaScript tabanlı dinamik oyun mantığı.

LocalStorage ile yüksek skor takibi.

CSS animasyonları ve interaktif UI.
